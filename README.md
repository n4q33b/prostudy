# ProStudy

AI-powered learning app for Maharashtra & CBSE board students (Class 5-10).

## Features

- Interactive HTML/CSS/JS simulations for each chapter
- AI Chat Tutor powered by Gemini 2.5 Flash
- Multimodal input (text, voice, image)
- Multi-board support (Maharashtra, CBSE)
- Multi-language support (Marathi, Hindi, English, Urdu)

## Tech Stack

- **Frontend:** Flutter
- **Auth:** Firebase Auth
- **Database:** Supabase (PostgreSQL + pgvector)
- **Storage:** Cloudflare R2
- **AI:** Google Gemini 2.5 Flash

## Project Structure

```
prostudy/
├── app/              # Flutter mobile app
├── content/          # Content generation scripts
├── infrastructure/   # IaC configs
├── simulations/      # HTML/CSS/JS simulations
├── scripts/          # Utility scripts
└── docs/             # Documentation
```
